# Snake-Game
A Snake Game using Pygame in Python 3 (both Snacks and Supersnacks included!)


Rules:

#1 Eat the brown snack for 1 point and the green supersnack for 2 points (be quick or it will disappear)

#2 Don't bite your tail

#3 Have fun!
